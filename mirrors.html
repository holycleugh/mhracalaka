Glowing nixshoes of the red churchbird.

Written herein is an unthorough Deerhouse of the most widespread and folkish Bondings;
    As beheld by Wedny, the Thiefsbestower, the Blissbringer, the Wordsteller.

Intokens.
The Mappings: the Motemappings, the Ropemappings, the Shapeshiftings, the twofold Workings.
The twofold Bondings.

The Mappings, those Bondings of each Thing to one Thing.

What is it like to be onemapping?
    „Each Thing maps to each Thing.“

What is it like to be overmapping?
    „Each Thing is mapped by each Thing.“

A twinmapping is a onemapping and overmapping Mapping.
    „Each Thing maps to and is mapped by each Thing.“
Or „Each Thing flips to and is flipped by each Thing.“


What is it like to be shrunken?
    „To be so, is to have a framing Threshold.“

What is it like to be steady?
    „To be so, is to have a wending Berth.“

What is it like to be shatterworthy?
    „To be so, is to have a matching Shard.“

What is it like to be gatherworthy?
    „To be so, is to have a matching Gard.“

What is it like to be evershattering?
    „To be so, is to have unending Shards.“

What is it like to be rungclimbing?
    „To be so, is to have a meeting Rungedrow.“


The Motemappings, those Mappings that beget the Allworld.

What is it like to be a Laddermapping?
    „Things grow as Rungtells grow.“

And if its Rungtell is Nought, then it is a Flatmapping.
And if its Rungtell is One, then it is a Oncemapping.
And if its Rungtell is Two, then it is a Twicemapping.
And if its Rungtell is Three, then it is a Thricemapping.
And if its Rungtell is over Three, then it is a Greatmapping.
And if its Rungtell is under Nought, then it is a Bowsmapping.
And if its Rungtell is broken, then it is a Bitmapping.

And if its Rungtell is everlasting, then it is a Rungedrow.

A Ropemapping is a Motemapping that is not a Laddermapping.

What is it like to be a Ringsnookmapping?
    „Things grow as the Onewedge ingirdles the Onering.“

What is it like to be a Bowsnookmapping?
    „Things grow as the Onewedge outgirdles the Onebows.“

What is it like to be a Mightmapping?
    „Things grow as Things grow.“

What is it like to be a Blightmapping?
    „Things grow as Things ungrow.“


The Shapeshiftings, those Mappings between Bodies.

What is it like to be shapeshifting?
If it is onemapping, then it is oneshifting.
And if it is overmapping, then it is overshifting.
And if it is twinmapping, then it is samefulshifting.
And if it is selfmapping, then it is selfshifting.
And if it is selfmapping and forbending, then it is innershifting.
    „Shifted Melds are Melded Shifts.“


The twofold Workings, those Mappings of two Things of Many to one Thing of that Many.

What is it like to be enclosing?
    „A Twain makes a Thing.“

What is it like to be freeyoking?
    „A Twain and a Thing, makes a Thing and a Twain.“

What is it like to be freeswapping?
    „A Thing and some Thing, makes that Thing and the Thing.“

What is it like to be oneholding?
    „A Thing and One, makes the Thing.“

What is it like to be flipflopping?
    „A Thing and some Thing, makes One.“

What is it like to be fairdealing?
    „A Thing with a Twain of Things, makes a Twain in Twains of Things.“



The twofold Bondings, those Bondings of one Thing to one Thing.

What is it like to be selfglancing?
    „Every Thing bonds with itself.“

What is it like to be forbidding?
    „No Thing bonds with itself.“

What is it like to be withmeting?
    „Every Thing which bonds with some Thing, is bonded to that Thing.“

What is it like to be thoroughing?
    „Every Thing which does not bond with some Thing, is bonded to that Thing.“

What is it like to be overshifting?
    „Every Thing which bonds with some Thing, bonds with every Thing which is bonded to that Thing.“

What is it like to be farknitting?
    „Every Thing which bonds with some Thing, bonds with every Thing which bonds with that Thing.“

What is it like to be everrowing?
    „Every Thing bonds with some Thing.“

What is it like to be ingrounding?
    „Some Thing bonds with no Thing.“

An evenworthing is a selfglancing, withmeting, overshifting Bonding.
    „This Thing and that Thing are evenworthy.“

A somefollowing is a selfglancing, unwithmeting, and overshifting Bonding;
And if it is thoroughing, then it is forfollowing;
And if it is ingrounding, then it is infollowing.
    „This Thing follows that Thing; that Thing foregoes this Thing.“
